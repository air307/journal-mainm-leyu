# journal-mainm-leyu

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖日常记录、资料整理以及一些静态页面的备份，不针对任何特定网站或域名。所有页面均为纯前端实现，无需后端服务支持。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面，命名遵循 `yyyy-mm-description` 或简短英文标识。
- `assets/`：按页面或功能组织，避免资源冗余。

## 页面归档说明

- 所有页面均为静态 HTML，可使用浏览器直接打开。
- 页面之间无依赖关系，可独立访问。
- 部分页面可能引用外部 CDN 资源（如字体、图标库），请确保网络通畅。
- 页面内容仅供学习、整理或备份用途，不构成任何形式的服务或承诺。

## 维护说明

- 本仓库不定期更新，新增页面会以独立文件形式加入。
- 历史页面通常不会删除，但可能因内容调整而更新。
- 若需引用或分发，请保留页面原始信息。
- 如有疑问或建议，欢迎提交 Issue 或 Pull Request。
